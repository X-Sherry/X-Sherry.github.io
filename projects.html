US Domestic Flight Price Prediction

2026 · View on GitHub

flight_price_prediction.ipynb 
import duckdb
import xgboost as xgb

# Connect to 82M+ records on disk
db = duckdb.connect('expedia_flights.db')
query = "SELECT * FROM flights WHERE cabin = 'economy'"
df = db.execute(query).df()

print(f"Data loaded. Shape: {df.shape}")
>>> Data loaded. Shape: (82410213, 14) Successfully!

This project aims to predict U.S. domestic flight prices using machine learning and a large-scale Expedia dataset containing over 82 million records. It explores the key factors that influence airfare, including booking timing, airline, cabin class, and itinerary characteristics, through statistical analysis and regression modeling. The project also demonstrates scalable data processing techniques using DuckDB to efficiently analyze large datasets that exceed in-memory computing limits.
